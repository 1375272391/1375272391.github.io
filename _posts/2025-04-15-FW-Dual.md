### ping测试PC-->Server
```CLI
PC>ping 17.0.0.254
```

在ping 完成后
```CLI
<USG6000V1>dis firewall session table 
```
随后断开其中一条线路 
例如R1 g0/0/0
```CLI
[Huawei-ospf-1]int g0/0/0
[Huawei-GigabitEthernet0/0/0]shu
```

随后再次ping测试
```CLI
PC>ping 17.0.0.254
```

再次查看
```CLI
<USG6000V1>dis firewall session table 
```
